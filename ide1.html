Bienvenue sur Brackets

Avec une prise en charge de préprocesseurs et des outils visuels ciblés, Brackets est un éditeur de texte moderne qui facilite la conception dans le navigateur. Il est intégralement conçu pour répondre aux besoins des designers web et des développeurs front-end.

Pourquoi utiliser Brackets ?

À la fois léger et puissant, Brackets est un éditeur de texte moderne. Par l'intégration d'outils visuels directement dans son interface, vous bénéficiez de la quantité d'aide adéquate lorsque vous le souhaitez, sans entraver votre processus créatif. Vous adorerez écrire du code dans Brackets.

Éditeurs en ligne

Au lieu de basculer entre des onglets de fichier, Brackets vous permet d'ouvrir une fenêtre dans le code qui vous intéresse le plus. Vous souhaitez travailler sur le CSS qui s'applique à un ID spécifique ? Placez le curseur de votre souris sur cet ID, appuyez sur Commande / Ctrl+E et Brackets vous montrera tous les sélecteurs CSS dotés de cet ID dans une fenêtre en ligne afin que vous puissiez travailler sur votre code côte à côte sans fenêtre contextuelle.

Aperçu en direct

Obtenez une connexion en temps réel à votre navigateur. Apportez des modifications aux codes CSS et HTML et vous verrez instantanément ces changements à l'écran. Examinez également l'emplacement où est appliqué votre sélecteur CSS dans le navigateur en plaçant simplement votre curseur dessus. Vous bénéficiez ainsi de la puissance d'un éditeur de code, tout en profitant des avantages pratiques d’un outil de développement dans le navigateur.

Prise en charge de préprocesseurs

Travaillez avec des préprocesseurs d'une toute nouvelle manière. Nous savons à quel point les préprocesseurs sont importants pour votre flux de travaux. C'est pourquoi nous voulons faire de Brackets le meilleur éditeur de code pour les préprocesseurs. Grâce à Brackets, vous pouvez utiliser la fonction Edition rapide et la mise en évidence en direct avec vos fichiers LESS et SCSS, ce qui rend plus facile que jamais le travail sur ceux-ci.
